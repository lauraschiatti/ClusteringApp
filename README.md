# CLusteringApp

Aplicación para clusterizar estudiantes con conceptos débiles similares utilizando diferentes técnicas. Trabajo de Grado de estudiantes de la Universidad Tecnológica de Bolívar.

## Tecnológias usadas

- Django (Python)
- CSS y JS
- HTML5
